## Getting started

1. Paste your genius.com API key into a file called `.genius_key`
1. Create a python3 virtualenv: `virtualenv -p python3 venv`
1. Activate the virtualenv: `source venv/bin/activate`
1. Install dependencies from `requirements.txt`: `pip install -r requirements.txt`
1. Run the project: `python3 index.py`
